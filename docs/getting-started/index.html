Começando

Obrigado por começar este curso. Para instalar vein tem que instalar primeiro R desde https://cran.r-project.org/. R pode ser instalado e muitos tipos diferentes de sistemas operacionais.

IDEs como RStudio o VSCode

É altamente recomendado instalar Rstudio ou VSCode.

Windows

Instala Rtools, que provee MinGW-w64 32/64-bit para compilar C, Fortran e C++. Isto é importante porque vein tambem tem codigo em Fortran.
