Começando

Instalação

A forma mas facil de instalar vein é, abrir R e escrever na termina:

install.packages("vein")

O a versão de GitHub

#install.packages("remotes")
remotes::install_github("atmoschem/vein")

De forma automática vão ser instalados os pacotes extras que são necessarios. Os pacotes necessarios sao:

  • sf, para trabalhar com vetores espaciais.
  • sp, antigo pacote de vetores espaciais, usado um caso específico de compatiilidade.
  • data.table processador de data.frame ultra rapido.
  • eixport: lee as saidas espaciais do VEIN e outros para gerar dados de entrada de modelos de qualidade do ar, como WRF-Chem.

[Pre-requisitos]

sf é um pacote de R com interace para as librerías GDAL (vetores), PROJ e GEOS. Esto implica que estas librerías tem que estar presentes no sistema primeiro antes de instalar sf. eixport importa funcções do ncdf4, então também é necessario instalar NetCDF.

As recomendações da pagina web de GitHub sf para instalar as librerias em diferentes siwtemas são:

Windows

Instala [R], ve a versão e instala Rtools, que provee MinGW-w64 32/64-bit para compilar C, Fortran e C++. Isto é importante porque vein tambem tem codigo em Fortran.

Ubuntu

sudo apt-get install libudunits2-dev libgdal-dev libgeos-dev libproj-dev libnetcdf-dev

Fedora

sudo dnf install gdal-devel proj-devel proj-epsg proj-nad geos-devel udunits2-devel netcdf-devel

Arch

pacman -S gdal proj geos netcdf
pacaur/yaourt/whatever -S udunits

MacOS

brew install pkg-config
brew install gdal netcdf
